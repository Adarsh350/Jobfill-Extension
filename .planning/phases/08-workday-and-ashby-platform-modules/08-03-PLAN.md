getJobDetails() (from RESEARCH.md code example):
```javascript
function getJobDetails() {
  var jobTitle = '';
  var companyName = '';
  var h1 = document.querySelector('h1');
  if (h1) jobTitle = h1.textContent.trim();
  // Tiered title parse: (1) "at X" regex, (2) "| X" split, (3) empty fallback
  var title = document.title || '';
  var atMatch = title.match(/\bat\s+(.+?)(?:\s*[|\-–]|$)/i);
  if (atMatch) companyName = atMatch[1].trim();
  else {
    var pipeParts = title.split(/[|–\-]/);
    if (pipeParts.length >= 2) companyName = pipeParts[1].trim();
  }
  return { companyName: companyName, jobTitle: jobTitle };
}
```

Pitfall: Ashby title parse may fail — implement tiered fallback (at X -> pipe split -> empty string). See RESEARCH.md Pitfall 4.
