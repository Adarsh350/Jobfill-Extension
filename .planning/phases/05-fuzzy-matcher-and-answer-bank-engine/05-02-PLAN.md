Group 4: scoreAnswerBankEntry
Jaccard implementation:
  function jaccard(setA, setB) {
    const intersection = new Set([...setA].filter(x => setB.has(x)));
    const union = new Set([...setA, ...setB]);
    return union.size === 0 ? 0 : intersection.size / union.size;
  }
Scoring:
  questionTokens = new Set(extractKeywords(questionText))
  entryTokens = new Set([...entry.keywords, ...extractKeywords(entry.question)])
  jaccardScore = jaccard(questionTokens, entryTokens)
  categoryBonus = infer category from questionText; 1.0 if matches entry.category, else 0
  score = Math.min(1.0, 0.8 * jaccardScore + 0.2 * categoryBonus)
Category inference keyword map (simple lookup, not fuzzy):
  salary/compensation/pay → 'salary'
  work.*auth/visa/permit/authorization/legally → 'work_auth'
  experience/background/worked → 'experience'
  motivat/interest/why.*work/why.*join/passion → 'motivation'
  skill/tool/proficien/familiar → 'skills'
  location/relocat/remote/hybrid → 'location'
  notice/available/start/when.*start → 'availability'
  diversity/equal/inclusive → 'diversity'
  cover.letter/introduce/about yourself → 'cover_letter'
  (default) → 'other'
Return 0 if both sets are empty.
Score must never exceed 1.0.
