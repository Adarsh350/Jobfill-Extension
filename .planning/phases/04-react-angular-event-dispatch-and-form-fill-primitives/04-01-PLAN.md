  <files>tests/helpers/dom-mock.js</files>
  <action>
Create `tests/helpers/dom-mock.js` that exports a `setupDom()` function and re-exports mock constructors. Node.js has no DOM; this shim provides minimal stubs sufficient to test events.js and filler.js logic.

The mock must provide:

1. `MockElement(tag, type)` — constructor producing an object with:
   - `tagName` (uppercase of tag), `type` (string), `checked` (bool), `value` ('')
   - `options` array (for select: each option has `.value` and `.text`)
   - `children` array, `shadowRoot` (null by default)
   - `isContentEditable` (false)
   - `_events` object for tracking dispatched events
   - `dispatchEvent(event)` — pushes `{ type: event.type, bubbles: event.bubbles }` onto `_events[event.type]`
   - `click()` — sets `this._clicked = true`
   - `querySelector(sel)` — returns first child where `child._selector === sel`, else null
   - `querySelectorAll(sel)` — returns all children matching; for shadow tests, children array can include nested MockElements

2. `MockEvent(type, options)` — constructor with `.type`, `.bubbles` (default false), `.composed` (default false)

3. `MockFocusEvent(type, options)` — same shape as MockEvent

4. `MockMutationObserver` — constructor accepting callback; exposes:
   - `observe(target, options)` — stores target/options, sets `this._observing = true`
   - `disconnect()` — sets `this._observing = false`
   - `_trigger()` — calls the stored callback (for test use: simulate a DOM mutation firing)

5. `setupGlobals()` — assigns onto `global`:
   - `HTMLInputElement = { prototype: { value: '' } }` with a writable value descriptor that tracks setter calls via `global._nativeSetterCallLog = []`
   - `HTMLTextAreaElement = { prototype: { value: '' } }` similarly
   - `Event = MockEvent`
   - `FocusEvent = MockFocusEvent`
   - `MutationObserver = MockMutationObserver`

For the native setter spy: use `Object.defineProperty` on `HTMLInputElement.prototype` and `HTMLTextAreaElement.prototype` to install a setter that pushes `{ el, value }` onto `global._nativeSetterCallLog` AND sets `el.value = value` on the mock element.

Export: `module.exports = { MockElement, MockEvent, MockFocusEvent, MockMutationObserver, setupGlobals }`.
